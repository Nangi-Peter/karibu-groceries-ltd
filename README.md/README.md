# Karibu Groceries LTD - Backend API

## Overview
Complete backend API for Karibu Groceries LTD management system built with Node.js, Express, and MongoDB.

## Features
- 🔐 JWT Authentication with role-based access control
- 👥 User management (Director only)
- 📦 Product/inventory management
- 📝 Procurement recording
- 💰 Cash sales processing
- 💳 Credit sales with payment tracking
- 📊 Aggregated reports (Director only)
- 🛡️ Security features (rate limiting, helmet, CORS)
- ✅ Input validation and sanitization

## Prerequisites
- Node.js (v14 or higher)
- MongoDB (v4 or higher)
- npm or yarn
